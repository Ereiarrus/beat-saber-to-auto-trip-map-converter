# Conversion Format Reference

## Beat Saber Note Format

The [BSMG Wiki](https://bsmg.wiki/mapping/map-format/beatmap.html) explains the meanings of Beat Saber note format:

```json
{
    "b": 5,
    "x": 2,
    "y": 0,
    "a": 0,
    "c": 1,
    "d": 1
}
```

Where:
- `b`: Beat time
- `x`: X position (lane)
- `y`: Y position (layer)
- `a`: Type (note type)
- `c`: Color (cut direction)
- `d`: Direction
